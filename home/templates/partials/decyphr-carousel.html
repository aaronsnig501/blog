{% trans 'Project:' %} Readr

{% trans 'Readr is intented to assist language learners with reading books written in their new language. The idea is that a new language learner translates phrases from books that they don\'t understand and Readr translates the text, creates an audio sample and provides a syntactical breakdown of the text and enables learners to practice the phrases they have translated' %}

{% trans 'Description:' %}

{% trans 'Readr is written in Django and NextJS and hosted on EC2 and Netlify respectively. All text is translated using Google Translate and Google\'s Language service to get a breakdown of the text to enable learners to see which words are adjectives, verbs, nouns, etc. and Google Books is used to enable students to manage their library to keep track of what they are reading' %}

{% trans 'Project:' %} i18nr

{% trans 'I was working with translations and had configured APIs for Google Translation and manually manually copying and pasting code into Google Translate which was time consuming and error-prone. i18nr takes text or a file as a parameter and translates it into the target languages' %}

{% trans 'Description:' %}

{% trans 'i18nr is written in Typescript using Heroku\'s oclif with a Django backend hosted on EC2 and the tool is available on NPM for installation within any project and is configurable to work as an npm command to translate your JSON, YAML or PHP files' %}
